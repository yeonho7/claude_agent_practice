# 🎉 Git 실습 저장소

## 웃긴 Git 농담들 😂

**Q: 왜 개발자는 화장실에 가는 걸 좋아할까요?**
A: Git commit을 하기 전에 한 번 더 생각하고 싶으니까요! 그래, 화장실에서 깨달음이 온다고! 💡

**Q: 당신의 코드가 안 돌아가면?**
A: 커밋 메시지를 보세요! 아마도 "빠른 수정" 또는 "이건 뭐하는 거야?"라고 적혀있을 겁니다. 🤦 
(이런 메시지를 본 당신의 미래 자신을 위해 기도하세요!)

## 꿀팁 🔥
- 절대 `git push --force`를 사용하지 마세요! (특히 다른 사람이 있을 때)
- `git revert`는 당신의 친구입니다 (undo의 또 다른 방법)
- merge conflict? 그건 일종의 게임입니다! 누가 이기나 봅시다! 🎮
